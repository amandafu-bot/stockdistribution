# Stock Distribution Tool (MVP)

## 🎯 產品目標
建立一個多倉庫電商庫存分配工具，協助品牌每月自動分配庫存到不同渠道。

## 🔷 核心流程
Warehouse → Channel → Strategy → SKU → Allocation → Export

## 📦 核心功能
1. 多倉庫庫存上傳
2. 倉庫管理
3. 渠道管理
4. 分配策略
5. SKU指派
6. 計算
7. 匯出

## 📊 Prototype
(Base44 demo 連結)

## 🧮 計算邏輯
1. 取得 SKU 在各倉庫庫存
2. 計算渠道可用庫存
3. 套用策略
4. 輸出結果

## 🚀 MVP Scope
- 多倉庫
- 渠道選倉
- 策略
- SKU分配
- 匯出
